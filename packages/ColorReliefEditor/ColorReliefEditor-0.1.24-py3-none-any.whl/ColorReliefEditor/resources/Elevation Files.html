This tab manages settings for digital elevation files and configures the Coordinate Reference System (CRS). It will combine multiple elevation files if more than one file is specified.

The elevation file(s) are used for generating the Hillshade, Color image, and final Relief image.

Settings

  • Elevation Files: This will be set when you drag a file to the blue box, or you can manually update it.
  • You will need to run in expert mode to change the Coordinate Reference System from the default of epsg:3857.

Instructions

Download Buttons: These buttons will bring you to various websites for downloading digital elevation files.

Once the files are downloaded, drag and drop them into the blue box. This will copy them to the project folder and automatically update the Elevation Files setting.

Download sites

Download (EarthExplorer), select Digital Elevation under the Data Sets tab and choose GMTED2010. Use the gmted_mea file.

Expert Settings

In expert mode, you can have multiple layers with different elevation files.

The layers all share the same rendering settings

The elevation files for a layer should be contiguous and must have the same resolution. You can use separate layers for:
1) different resolutions (high vs low) or
2) for discontinuous areas. Layer names cannot have a space in them.

  • Active Layer: Select the layer to use for preview and full build.
  • Layer Name: Specify the name for this layer.
  • Source: (Optional) Provide the source of the elevation files.
  • License: (Optional) Specify licensing information for the elevation files.
